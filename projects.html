Implementing RL policies for Autonomous Vehicles

CARLA is a car-driving simulation software that has been developed from the ground up to support development, training, and validation of autonomous driving systems. My B.Tech Project (BTP) is to use reinforcement learning to train various maneuvers in autonomous systems, like right/left turn, going straight, exiting a roundabout, etc. I have already trained a maneuver, using deep Q-learning, in which the car has to go straight till it reaches a checkpoint and then has to turn right to reach its destination.

WMG Internship

My task for this project was to convert a text file written in SDL grammar and convert it into OpenDrive (.xodr) format, which can be plugged into various simulation softwares like CARLA, RoadRunner, etc. SDL grammar allows user to describe road networks in a very simplified manner, unlike the XODR format. Hence, a program that converts SDL to OpenDrive will be invaluable to users who want to describe road networks efficiently and accurately.
